how to back up and restore a mysql database

MySQL dump

使用mysqldump command,这条命令连接数据库，然后创建一个SQL dump 文件。这个dump文件包含了重新建库的所需SQL语句。正确的语法是这样的:

$ mysqldump --opt -u [uname] -p[pass] [dbname] > [backupfile.sql];
